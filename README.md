## 🏆 Featured Projects

### 🔹 Projexis – Student Project Manager
A comprehensive full-stack application designed to streamline student project management, tracking, and collaboration.
- **Tech Stack:** ASP.NET Core, React, SQL Server
- **Architecture:** Built using **Clean Architecture** with a robust database design containing **15+ relational tables**.
- **Security:** Secured with **JWT Authentication** and **Role-based access control (RBAC)**.

### 🔹 LabFlow - Smart Lab Management System
Comprehensive laboratory management system for educational institutions to manage labs, schedules, and resources.
- **Key Features:** Role-based dashboards for Super Admin, Faculty, etc., Advanced lab scheduling with time-conflict resolution.
- **Tech Stack:** React, Supabase, Bootstrap

### 🔹 SmartBill Pro - Multi-Tenant Invoice System
Production-ready multi-tenant invoice management platform with advanced analytics.
- **Key Features:** Advanced analytics dashboard with revenue charts, Smart invoice generation with GST calculation.
- **Tech Stack:** React 18, Vite, Supabase

### 🔹 CodeCrafts Academy
Interactive learning platform for web development fundamentals (HTML, CSS, JS, Git).
- **Key Features:** Structured learning modules, Interactive lessons.
- **Tech Stack:** React, JavaScript, Tailwind CSS

### 🔹 MOM - Minutes of Meeting
Web application for managing organizational meetings and attendance.
- **Key Features:** Complete CRUD operations with 36+ stored procedures, Implemented meeting scheduling with conflict resolution.
- **Tech Stack:** ASP.NET Core MVC, C#, SQL Server

### 🔹 React Recipe App
Search and discover recipes with a modern responsive UI.
- **Key Features:** API-based recipe fetching, Search & Filter mechanics.
- **Tech Stack:** React, REST APIs, CSS3

### 🔹 YouTube Clone (Backend)
Video streaming platform replica backend architecture.
- **Key Features:** Video upload & streaming, User authentication.
- **Tech Stack:** Node.js, Express, MongoDB
